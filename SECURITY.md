# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅ Yes    |

## Reporting a Vulnerability

Please do NOT report security vulnerabilities through 
public GitHub issues. This protects users while we fix it.

**Email: saif25.sayyed@gmail.com**

**Subject line:** [SECURITY] Brief description

We will respond within 48 hours and provide a fix 
within 7 days for critical issues.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes

We will credit you in the release notes when the fix ships.
